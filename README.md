
## 数据结构与算法学习

收集分布式系统相关的数据结构与算法，这些代码文件均取自成熟开源的项目(如Leveldb,Oceanbase,TFS,tera,toft等)，主要学习：

1. 数据结构、算法的优缺点，适用场景。

2. 代码实现技巧，如何优化代码性能。

3. 如何做测试：功能测试、性能测试。



## 参考
《大数据日知录 - 架构与算法》  
《大规格分布式存储系统 - 原理解析与架构实战》
